# 🎓 Student Grade Analyzer

A lightweight data analysis project built with Python and NumPy that processes student grades to extract meaningful insights such as performance classification, averages, and statistical summaries.

## 🚀 Features
- 📊 Calculate average, max, and min scores
- ✅ Identify passed and failed students
- 🌟 Extract top-performing students
- 🧠 Classify students based on performance
- ⚡ Fast computations using NumPy



## 🧠 Tech Stack
- Python 🐍
- NumPy 📊



## 📂 Project Structure
